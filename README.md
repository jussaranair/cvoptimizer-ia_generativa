# CV Optimizer

Aplicação para otimizar currículos (CV) com interface web usando Streamlit e armazenamento local em SQLite.

## Funcionalidades

- Interface interativa com Streamlit para enviar, analisar e melhorar CVs.
- Armazenamento local de sessões e histórico em banco SQLite.
- Hooks para integrar modelos ou regras de otimização (configurável).

## Requisitos

- Python 3.8+ (recomendado 3.10+)
- Streamlit
